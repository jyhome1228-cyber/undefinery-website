# Undefinery Solution Website

순수 HTML, CSS, JavaScript로 구성된 다중 페이지 정적 웹사이트입니다.

## Structure

- `index.html`
- `about.html`
- `solutions.html`
- `portfolio.html`
- `production.html`
- `insights.html`
- `contact.html`
- `portfolio/*.html` — 프로젝트 상세
- `insights/*.html` — 인사이트 상세
- `assets/css/style.css`
- `assets/js/common.js`

별도 빌드 없이 `index.html`을 열거나 GitHub Pages로 배포할 수 있습니다.
