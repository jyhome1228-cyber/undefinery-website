# Undefinery Solution Website

Official website concept for Undefinery Solution — a growth architecture company connecting strategy, design, production, business systems and ongoing brand operation.

## Current version

- Cursor-reactive interactive hero
- About and core positioning
- Six solution areas
- Portfolio categories
- AI Production System
- Insights and contact sections
- Responsive desktop/mobile layout

## Run locally

Open `index.html` directly, or run any static file server from the project root.

## Structure

- `index.html` — page content and semantic structure
- `styles.css` — responsive visual system
- `script.js` — glow grid interaction
